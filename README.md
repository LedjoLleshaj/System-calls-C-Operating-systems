# System-calls-C-Operating-systems
Client-Server written using only Linux System calls in C

## Documentation

You can find the documentation generated by Doxygen [here](https://github.com/LedjoLleshaj/System-calls-C-Operating-systems/blob/main/documentation/html/md_README.html).


## How to compile:

```bash
make
```

## How to run
You need two terminals opened at the same time
```./client_0 <path>```
```./server ```


## How to terminate Client
1. Open a another terminal
2. Execute the command below:

    ```bash
    kill -s SIGUSR1 $(pgrep client_0)
    ```
